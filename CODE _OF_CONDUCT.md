
code of Conduct

This repository is part of the QuietWire AI Training Program.  
All contributors, reviewers, and learners are expected to follow this Code of Conduct.

## 1. Respect

- Be kind, patient, and professional in all interactions.
- No harassment, hate speech, or personal attacks.
- Assume good intent, but give clear and constructive feedback.

## 2. Learning Environment

- This is a training repository. Mistakes are welcome as part of learning.
- Ask questions politely and help others when you can.
- Use clear language and avoid sarcasm in written communication.

## 3. Security & Confidentiality

- Do not share real customer data, secrets, or private credentials.
- Examples must use fake or anonymized data only.
- Follow any internal QuietWire security policies when applicable.

## 4. Reporting Issues

If you see behavior that violates this Code of Conduct:

- Open a private issue (if appropriate), or  
- Contact the maintainer/mentor supervising this repository.

## 5. Enforcement

Maintainers may warn, restrict, or remove participation from anyone who violates this Code of Conduct.

By contributing or using this repository, you agree to follow these rules.



# Guide de Contribution – QuietWire AI

Merci de votre intérêt pour ce projet. Pour contribuer efficacement,
veuillez suivre les instructions suivantes :

## 1. Structure des Fichiers
- Utiliser des noms de fichiers clairs et cohérents.
- Respecter l’organisation du dépôt (EN / AR / FR / ES).

## 2. Style et Qualité
- Fournir un contenu structuré et facile à lire.
- Écrire dans un français correct, formel et professionnel.

## 3. Processus de Contribution
- Créer une branche dédiée avant toute modification.
- Soumettre une Pull Request avec une description claire.
- Mentionner les superviseurs si nécessaire.

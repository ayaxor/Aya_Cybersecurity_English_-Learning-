

# Support et Assistance – QuietWire AI

Si vous avez besoin d’aide concernant :
- La compréhension des documents
- L’organisation du dépôt
- Les fichiers techniques ou linguistiques

Veuillez suivre les étapes :

## 1. Vérifier la Documentation
Lire attentivement les sections EN / AR / FR avant de poser une question.

## 2. Contacter le Support
Pour toute difficulté, envoyer un message :
- Avec une description claire du problème
- Avec un exemple ou une capture si possible

## 3. Temps de Réponse
Le support répond généralement dans un délai raisonnable,
selon la charge de travail.

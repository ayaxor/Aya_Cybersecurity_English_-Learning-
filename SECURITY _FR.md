

# Politique de Sécurité – QuietWire AI

La sécurité est une priorité essentielle dans ce projet.  
Toutes les actions doivent respecter les règles suivantes :

## 1. Protection des Données
- Ne partager aucune information sensible ou interne.
- Vérifier toute source avant d’ajouter du contenu.

## 2. Intégrité des Fichiers
- Ne pas modifier les documents critiques sans validation.
- Maintenir une documentation propre et sécurisée.

## 3. Signalement des Vulnérabilités
Si vous découvrez un problème ou une faille :
- Signalez-le immédiatement au superviseur technique.
- Ne tentez pas de le corriger seul sans autorisation.





# آية جمال  
### مستودع تعلم الأمن السيبراني واللغة الإنجليزية

هذا المستودع يمثل مسار تعلمي الشخصي في مجال الأمن السيبراني وتطوير اللغة الإنجليزية.

يحتوي المستودع على:
- ملاحظاتي وتلخيصاتي
- تقدّم دراستي في الأمن السيبراني
- تقدّم تطوير لغتي الإنجليزية
- روابط ومصادر خارجية

- توثيق مشواري العلمي كمحترفة



# Code de Conduite – QuietWire AI

Afin de maintenir un environnement professionnel, respectueux et productif,
tous les contributeurs doivent respecter les règles suivantes :

## 1. Respect Mutuel
- Utiliser un langage poli et professionnel.
- Éviter les propos offensants, discriminatoires ou agressifs.

## 2. Professionnalisme
- Fournir un travail honnête et rigoureux.
- Suivre les instructions données par les superviseurs techniques.

## 3. Transparence
- Signaler toute erreur, problème ou difficulté rencontrée.
- Travailler de manière claire et organisée.

## 4. Sécurité
- Protéger les données, documents et informations confidentielles.
- Ne jamais partager de contenu interne sans autorisation.
